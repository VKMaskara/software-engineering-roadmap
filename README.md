# 💻 Software Engineering Roadmap

> **Uma jornada documentada rumo à Engenharia de Software.**

Este repositório documenta minha trajetória de aprendizado em Engenharia de Software, desde os fundamentos da programação até o desenvolvimento de sistemas completos.

Mais do que um conjunto de anotações, este projeto é uma base de conhecimento construída durante meus estudos, reunindo teoria, exercícios, desafios, projetos e documentações produzidos ao longo da minha formação.

---

# 🎯 Objetivo

Meu objetivo é construir uma base sólida em Engenharia de Software, compreendendo profundamente os fundamentos da computação antes de avançar para tecnologias específicas.

Ao final desta jornada pretendo ser capaz de:

- Desenvolver sistemas completos com autonomia;
- Resolver problemas utilizando pensamento computacional;
- Modelar soluções de software de forma eficiente;
- Utilizar Inteligência Artificial como ferramenta de produtividade, e não como dependência;
- Criar projetos reais aplicando boas práticas de desenvolvimento.
