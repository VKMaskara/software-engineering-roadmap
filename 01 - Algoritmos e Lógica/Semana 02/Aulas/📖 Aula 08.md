# Exercícios Integradores

### Essa é uma aula focada na pratica desenvolvida durante esta Semana

---
## 🎯 Objetivo da Aula

Ao final desta aula você será capaz de:

Resolver problemas utilizando todos os conceitos aprendidos até agora;
Identificar quais ferramentas utilizar para analisar um algoritmo;
Desenvolver soluções pensando como um programador;
Consolidar a base antes de aprender estruturas de repetição

---

# 📖 Introdução

-> Durante essa semana, estudei os fundamentos da lógica em componentes separados.

Aprendi:

- Algoritimo;
- Variáveis;
- Operadores;
- Estrutura de decição;
- Teste de Mesa;
- Fluxo de Execução;
- Depuração

Agora como todo bom Programador vamos juntar esses conhecimentos para solucionar um problema ainda maior.

- Agora com essa bagagem temos a capacidade de ao receber uma tarefa realmente pensarmos como um programador.

# 💡 Exemplo

-> Um bom programador não é o que corre fazer código, mas sim aquele que faz boas perguntas.

Imagine que esta criando m sistema para Barbearia:

As regras são:
- Usuario agenda horario
- Não pode ter conflito de horario
- Tem que ter os horarios disponiveis 

Isso vem muito antes de codar, Ser programador é entender que o Software vem de entender o problema e não ir para o códiogo primeiro.

---

# 📌 O mais importante
Um erro muito comum dos iniciantes é abrir o VS Code antes de entender o problema.

O correto é:

Entender

↓

Planejar

↓

Resolver

↓

Programar

A programação é apenas a última etapa.
