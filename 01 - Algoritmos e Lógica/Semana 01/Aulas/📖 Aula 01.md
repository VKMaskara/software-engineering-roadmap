# Aula 01 — O que é um Algoritmo?

## 🎯 Tema

**O que é um Algoritmo?**

### Objetivo da aula

Compreender o conceito de algoritmo, desenvolver a habilidade de decompor problemas em etapas e entender a base do pensamento computacional.

---

# 📖 Contexto

A principal habilidade que todo programador precisa desenvolver antes de realmente escrever qualquer linha de código é:

> **Resolver problemas.**

Um desenvolvedor é, acima de tudo, um resolvedor de problemas.

As linguagens de programação são apenas ferramentas utilizadas para transmitir instruções ao computador.

Assim como um arquiteto, o programador precisa:

- Entender a necessidade;
- Fazer um planejamento;
- Desenhar a "planta";
- Só então começar a construir.

---

# 🎯 Objetivos da Aula

Ao final desta aula, você deverá ser capaz de:

- Explicar o que é um algoritmo;
- Diferenciar algoritmo de programa;
- Entender o conceito de **Entrada, Processamento e Saída**;
- Resolver pequenos problemas utilizando apenas lógica;
- Escrever algoritmos em linguagem natural.

---

# 🧠 O que é um Algoritmo?

Um **algoritmo** é uma sequência finita de passos lógicos e organizados de forma definida para resolver um problema ou executar uma tarefa.

> **Uma forma lógica de resolver problemas.**

Todo algoritmo possui:

- Ordem;
- Início;
- Meio
- Fim;
- Instruções detalhadas.

---

## ☕ Exemplo

Imagine que você peça para um robô:

> **"Faça um café."**

Provavelmente ele não saberá o que fazer, pois essa instrução é muito genérica.

O correto seria fornecer instruções detalhadas:

1. Pegue uma xícara;
2. Pegue o pó de café;
3. Coloque água na cafeteira;
4. Ligue a cafeteira;
5. Aguarde o café ficar pronto;
6. Sirva o café na xícara.

Perceba que cada passo foi descrito em ordem.

---

# 💻 Como um computador pensa?

Na verdade, o computador **não pensa**.

Ele apenas executa instruções exatamente como foram informadas.

Imagine-o como um funcionário extremamente eficiente e obediente, que faz somente aquilo que foi solicitado.

Se faltar uma instrução, ele não irá "adivinhar" o que deveria fazer.

---

# 💡 Algoritmo × Programa

Embora pareçam a mesma coisa, existe uma diferença importante.

## Algoritmo

É a solução para um problema.

Não depende de nenhuma linguagem de programação.

Pode ser escrito em:

- Português;
- Pseudocódigo;
- Fluxograma.

O exemplo do café apresentado anteriormente já é um algoritmo.

---

## Programa

É a implementação de um algoritmo utilizando uma linguagem de programação.

Por exemplo:

- Java;
- JavaScript;
- Python;
- C++;
- C#.

Em outras palavras:

> Todo programa possui um algoritmo.

Mas:

> Nem todo algoritmo se torna um programa.

---

# 📦 Entrada → Processamento → Saída

Todo algoritmo passa por três etapas fundamentais.

## 📥 Entrada

Recebe os dados necessários.

↓

## ⚙️ Processamento

Realiza as operações necessárias utilizando os dados recebidos.

↓

## 📤 Saída

Exibe o resultado.

---

# 🌍 Exemplo utilizando o R.O.M.A.

### 📥 Entrada

O usuário envia o arquivo contendo os endereços.

↓

### ⚙️ Processamento

O sistema processa os dados e realiza a otimização das rotas.

↓

### 📤 Saída

O entregador recebe a rota otimizada.

---

# 📌 Resumo da Aula

Hoje aprendemos que:

- Um algoritmo é uma sequência ordenada de instruções;
- Todo algoritmo possui início, meio e fim;
- Algoritmos não dependem de uma linguagem de programação;
- Todo programa implementa um algoritmo;
- Todo algoritmo pode ser representado pelo modelo:

```text
Entrada
   ↓
Processamento
   ↓
Saída
```

## 💭 Conclusão

Mais importante do que decorar a definição de algoritmo é compreender que **programar significa transformar problemas em sequências lógicas de passos**.

Essa habilidade é a base de toda a programação e será utilizada em qualquer linguagem que você aprender no futuro.
