# 🧠 Módulo 01 — Algoritmos e Lógica de Programação

> "Antes de aprender uma linguagem de programação, é preciso aprender a pensar como um programador."

Este módulo reúne toda a minha jornada de aprendizado sobre os fundamentos da programação. Aqui estão documentadas as aulas, anotações, exercícios, desafios e revisões desenvolvidos durante minha formação em Engenharia de Software.

---

# 🎯 Objetivo do Módulo

Construir uma base sólida em lógica de programação, desenvolvendo a capacidade de resolver problemas antes de escrever qualquer código.

Ao concluir este módulo, serei capaz de:

- Compreender o conceito de algoritmo;
- Modelar problemas utilizando lógica;
- Trabalhar com variáveis e tipos de dados;
- Utilizar operadores aritméticos, relacionais e lógicos;
- Construir estruturas de decisão e repetição;
- Desenvolver algoritmos independentes de linguagem;
- Pensar como um desenvolvedor antes de programar.

# 📖 Metodologia

Este módulo não consiste apenas em assistir aulas.

Cada conteúdo segue o seguinte processo:

1. Aula teórica
2. Anotações em Markdown
3. Exercícios
4. Desafio
5. Correção
6. Revisão semanal (Sprint Review)
7. Documentação da evolução

O objetivo é aprender os conceitos profundamente, e não apenas memorizar sintaxe.
